(async, static) getMeshesForFilename(filename, color, localReader, sharedMeshes, sharedTextures, showUnmaterialed, callback)

Gets a mesh array from Model file and sends as argument to callback. Uses a cache of meshes in order to never read the same model file twice.

Parameters:
Name Type Description
filename Number

The fileId or baseId of the Model file to load
color Array

RGBA array of 4 integers
localReader LocalReader

The LocalReader object used to read data from the GW2 .dat file.
sharedMeshes Object

Value Object for keeping the texture cache.
sharedTextures Object

Value Object for keeping the texture cache.
showUnmaterialed boolean

If false does not render any models with missing materials.
callback function

Fired once all meshes have been loaded. three arguments are passed to the callback function.

The first argument is an Array with each textured THREE.Mesh objects.

The second argument is a boolean, true indicates that these meshes were not loaded from the dat file, but retrieved from the run time cache.

The third argument is the bounding spehere of this model file.
